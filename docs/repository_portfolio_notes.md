# Portfolio Notes

This repository is shaped for fast technical review:

- The first screen explains the problem, system, results, and how to run tests.
- Sample data is anonymized and small enough to clone.
- Large files are excluded: raw video, model weights, TensorRT engines, virtual environments, and calibration dumps.
- The Python package is installable with `pip install -e ".[dev]"`.
- Tests cover assessment logic, live trainer state, event logging, safety gates, C3D export, and projector target scoring.

The private working repository contains additional thesis drafts, lab captures, historical experiments, and presentation assets. Those are intentionally not part of this public portfolio snapshot.
